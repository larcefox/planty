# Planty Frontend

Vite + React + TypeScript фронтенд со стартовым layout и TailwindCSS.

## Команды

```bash
# установка зависимостей
npm install

# запуск dev-сервера на http://localhost:5173
npm run dev

# линтинг
npm run lint
```

Основной layout находится в `src/App.tsx`, заготовки компонентов — в `src/components/`.
